My Projects

  • Louisville Jazz Initiative

    This is the website for the Louisville Jazz Initiative, a non profit organization I co-founded in April 2021. We are a collective of jazz musicians in Louisville, KY committed to culturally authentic playing and teaching of jazz music. This is a brand new organization so the website lacks media; with many upcoming events though, that will soon change.

  • Fiber Coven

    This is the website for Fiber Coven, a weekly witchy themed podcast about the knitting, spinning, and everything fiber related. Check out the repo here.

  • Mastermind

    This is my take on the classic paper and pencil game: Mastermind. I built this to practice JS listeners and it quickly became my testing ground for git and GitHub. The game is officially functional which is very exciting! That said, it is still in an early phase and not styled in any way. Feel free to check it out, play to your hearts content (it's pretty addicting!) and offer any suggestions or general advice. Check out the repo here.

  • Daily Web Comic

    I made this page to learn more about manipulating iframe elements. It displays some of my favortie daily web comics. My initial plan with this page is to learn how to make iframes more responsive and with dynamic sizing. The long term goal is to include a tool that allows you to preview a new comic's site with a rectangle that can be moved over the comic, resized to crop the comic, and output the numbers needed to add that comic to the page. Check out the repo here.

  • My Music Page

    This is my landing page as a professional musician. I haven't updated content it in a couple years, let alone the style and functionality. I would like to do a full reboot of this page when I have time.

About Me

Chad O'Brien is a freelancer based in Louisville, Kentucky. Among his many interests are performing, teaching, and writing music, web development, urban farming, and social justice activism. He is currently the Event Coordinator for the University of Louisville School of Music, Operations Manager for the Louisville Jazz Initiative, and a student at Code Louisville. You can get to know Chad better by checking out the projects above or emailing him.
