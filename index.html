My Projects

  • My Music Page

    This is my most recent completed project - my landing page as a professional musician. It is a mobile-first, responsive site that uses semantic HTML5, CSS3, JavaScript, and PHP. Since I initially built this site as the final project of my Code Louisville Front End Web Development course, it's packed with fun JS scripts. Feel free to view the GitHub repo for this site and check out the README for a full list of design features.

  • Louisville Jazz Initiative

    This is the website for the Louisville Jazz Initiative, a non profit organization I co-founded in April 2021. We are a collective of jazz musicians in Louisville, KY committed to culturally authentic playing and teaching of jazz music. I built this website before knowing about mobile-first design, accessibilty standards, and CSS flexbox and am now in the process of updating it to reflect best practices.

  • Code Breaker

    This is my take on the classic paper and pencil game: Mastermind. I built this to practice JavaScript and it quickly became my testing ground for git and GitHub as well. The notebook paper styling has been fun to experiment with but needs a little more tweaking to be made more responsive. Feel free to check it out, play to your hearts content (it's pretty addicting!) and offer any suggestions. If you're interested in the code, you can check out the GitHub repo here.

  • Fiber Coven

    This is the website for Fiber Coven, a weekly witchy themed podcast about the knitting, spinning, and everything fiber related. Since most of my projects are self directed, I enjoyed this opportunity to work with an established brand. It has also been a fun challenge integrating third party widgets (a PodBean player, MailChimp newsletter sign up, etc.) into the design. If you want to explore how I did that, check out the GitHub repo here.

  • JavaScript 30 in 30

    This is a website to feature my interpretations of the #JavaScript30 project tutorials by Wes Bos, found on javascript30.com. I've only just started but am really enjoying these bite-sized tasks and learning a lot! Check out the tutorials and feel free to give the repo a look as well.

About Me

Chad O'Brien is a freelancer based in Louisville, Kentucky. Among his many interests are performing, teaching, and writing music, web development, urban farming, and social justice activism. He is currently the Event Coordinator for the University of Louisville School of Music, Operations Manager for the Louisville Jazz Initiative, and a student at Code Louisville. You can get to know Chad better by checking out the projects above or emailing him.
